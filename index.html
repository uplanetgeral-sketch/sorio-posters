Parâmetros URL universais

Conteúdo (todos os templates)

hero              URL da imagem hero (default: assets/hero-sample.png)
logo              URL do logo (default: assets/logo.png)
selo              URL do selo (default: assets/selo-default.png)
title             Título Oswald Bold. "|" = quebra de linha
info_top          Ingredientes, tracked caps
accent            Claim italic Cormorant
info_bottom       Footer tracked
format            'WxH' (1080x1350 default · 1080x1080 · 1080x1920)
display_size      Font-size px do título
accent_size       Font-size px do accent
overlay_strength  0.0 a 1.6, multiplica gradient (F02 only)
hero_position     CSS background-position
bg_color          Hex color do background (F06/F09/F10)
show_logo         1 | 0
show_selo         1 | 0
show_accent       1 | 0
show_info_top     1 | 0
show_info_bottom  1 | 0

Branding (logo + selo positioning)

logo_position     top-left (default) | top-center | top-right | bottom-left | bottom-right | hidden
logo_scale        0.6 a 1.4 (default 1.0)
selo_position     top-right (default) | top-left | top-center | bottom-left | bottom-right | over-hero | hidden
selo_scale        0.5 a 1.6 (default 1.0)

v1.9 Editor params (F09 only · em rollout para outros)

z_<layer>         z-index numérico (override CSS) — controla ordenação de layers
<layer>_color     CSS color override por layer de texto (paleta Blueprint)
<layer>_x         offset horizontal em px (default 0)
<layer>_y         offset vertical em px (default 0)
<layer>_rotation  graus de rotação (default 0)
<layer>_scale     multiplicador de escala (default 1.0)

<layer> ∈ { hero, title, accent, info_top, info_bottom, logo, selo }

Ex: ?z_hero=5&hero_x=24&title_color=%23D9C8A8&title_rotation=-3

Exemplo

f09.html?title=Aperol%7CSpritz&info_top=Aperol · Soda · Espumante&accent=Sem pressa. Com rio.&info_bottom=Valada do Ribatejo · sorio.pt&z_hero=5&hero_x=12
